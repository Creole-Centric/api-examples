🎙️ CreoleCentric TTS Widget

Embeddable Haitian Creole Text-to-Speech for Your Website

What is This?

The CreoleCentric TTS Widget is a drop-in JavaScript component that adds Haitian Creole text-to-speech capabilities to any website. Just copy a few lines of code, and your users can generate natural-sounding Haitian Creole speech directly on your site.

⚡ Zero Dependencies

No frameworks required. Just one JavaScript file.

💾 Smart Caching

Audio cached in browser for 7 days. Instant replay.

🎨 Customizable

Light/dark themes and custom styling options.

📱 Mobile Ready

Works perfectly on mobile and desktop browsers.

Quick Start (2 Steps)

Step 1: Add the Widget Container

<!-- Add this div where you want the widget to appear --> <div id="creolecentric-tts" data-cc-widget data-api-key="cc_your_api_key_here"></div>

Step 2: Include the Widget Script

<!-- Add this before the closing </body> tag --> <script src="creolecentric-widget.js"></script>
📝 Note: Replace cc_your_api_key_here with your actual API key from creolecentric.com

Configuration Options

You can customize the widget using data attributes or JavaScript initialization:

Option Data Attribute Default Value Description
apiKey data-api-key (required) Your CreoleCentric API key
voiceId data-voice-id i4mRPwKM2yHwXhbmkN514 Voice ID to use for synthesis
modelId data-model-id ccl_ht_v100 TTS model version
theme data-theme light Widget theme: 'light' or 'dark'

Example: Dark Theme with Custom Voice

<div id="my-tts" data-cc-widget data-api-key="cc_your_key" data-theme="dark" data-voice-id="your_voice_id"></div> <script src="creolecentric-widget.js"></script>

Advanced: JavaScript Initialization

For more control, you can initialize the widget programmatically:

<!-- HTML --> <div id="my-custom-widget"></div> <script src="creolecentric-widget.js"></script> <script> // Initialize after page load CreoleCentricWidget.init({ containerId: 'my-custom-widget', apiKey: 'cc_your_api_key_here', voiceId: 'i4mRPwKM2yHwXhbmkN514', modelId: 'ccl_ht_v100', theme: 'light', placeholder: 'Ekri tèks ou an kreyòl Ayisyen...' }); </script>

Features

🔄 Automatic Caching

Generated audio is automatically cached in the browser's localStorage for 7 days. If you enter the same text again, it plays instantly without re-generating.

🗑️ Smart Cache Invalidation

When you modify the text in the input field, the widget automatically clears the audio player and cache for that text, ensuring you always hear the latest version.

⏯️ Play/Pause Controls

The widget includes a built-in audio player with play/pause controls. The button dynamically updates based on the current state.

📊 Status Feedback

Users receive real-time feedback on job status: "Generating audio...", "Processing...", "Ready to play!", and error messages if something goes wrong.

Getting Your API Key

  1. Go to creolecentric.com/signup and create an account
  2. Navigate to Account Settings → API Keys
  3. Click "Create New API Key"
  4. Copy your key (it starts with cc_)
  5. Use it in the data-api-key attribute
💡 Tip: Keep your API key secure. Don't commit it to public repositories. Consider using environment variables or server-side rendering to inject the key.

Troubleshooting

Widget Not Appearing?

  • Check that the creolecentric-widget.js file path is correct
  • Verify the container div has a unique id attribute
  • Open browser console (F12) to check for JavaScript errors

"API key is required" Error?

  • Ensure data-api-key attribute is set
  • Verify your API key starts with cc_
  • Check that the API key is valid and not expired

Audio Not Playing?

  • Check your browser's console for network errors
  • Verify you have sufficient API credits
  • Try clearing localStorage: localStorage.clear() in console
