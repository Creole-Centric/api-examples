Voice Agent — Widget Embed

This is the minimum HTML you need to embed a CreoleCentric Voice Agent on any web page. Save this file, replace YOUR_AGENT_ID with the agent ID you got from create-agent.sh, and open the file in a browser.

Look for the floating Voice chat button in the bottom-right corner. Click it, grant microphone permission, and start talking.

Public agent required. The widget will only work for anonymous visitors if your agent was created with is_public: true. If you need authenticated-caller flows, generate a session token from your backend and use the widget's programmatic API instead — see the docs.

Configuration options

The following data-* attributes on the script tag customize the widget:

  • data-agent-id — required. The agent ID.
  • data-theme — light or dark. Default light.
  • data-position — bottom-right, bottom-left, top-right, top-left. Default bottom-right.
  • data-greeting — text shown at the top of the panel. Default "Voice chat".

Content Security Policy

If your site sets a CSP, add these directives:

script-src 'self' https://creolecentric.com;
connect-src 'self' https://creolecentric.com wss://creolecentric.com;
media-src 'self' blob:;
